## TalkFlow: AI 음성 플래너 웹 클라이언트

TalkFlow는 사용자의 일상 계획을 혁신하는 AI 기반의 스마트 플래너입니다. 음성을 통해 직관적으로 일정을 관리하고, 지능적인 위치 기반 동선 관리 기능으로 사용자의 시간을 최적화합니다. 이 레파지토리는 TalkFlow의 **웹 클라이언트**를 담당합니다.

### 주요 기능

-   **AI 기반 음성 일정 생성:** 자연스러운 대화를 통해 일정을 생성하고 관리합니다.
-   **스마트 지도 연동:** 일정에 포함된 위치 정보를 기반으로 최적의 동선을 제공합니다.
-   **지능형 알림 시스템:** 다음 일정까지의 이동 시간을 고려한 스마트 알림을 제공합니다.
-   **하이브리드 렌더링:** 사용자와의 상호작용이 많은 페이지는 CSR로, 초기 로딩이 중요한 페이지는 SSR로 효율성을 높였습니다.

### 기술 스택

-   **Next.js:** 프론트엔드 프레임워크
-   **React Query:** 서버 상태 관리
-   **Zustand:** 클라이언트 상태 관리
-   **Google Maps API:** 지도 연동

### 프로젝트 구조

이 프로젝트는 프론트엔드와 백엔드가 분리된 구조입니다.

-   **웹 클라이언트 (현재 레파지토리)**: TalkFlow의 사용자 인터페이스를 담당합니다.
-   **서버 API**: **`DOforTU/talkflow-server: That's for server about TalkFlow.`** 레파지토리에서 NestJS 기반의 백엔드 API를 제공합니다.

### 시작하기 (Getting Started)

프로젝트를 로컬 환경에서 실행하려면 먼저 서버를 구동해야 합니다.

1.  **서버 설정**: `talkflow-server` 레파지토리를 클론하고 `README.md`의 지침에 따라 서버를 실행합니다.
2.  **클라이언트 설치**: 이 레파지토리를 클론하고 의존성 패키지를 설치합니다.
    ```sh
    git clone [이 레파지토리 URL]
    cd ui
    npm install
    ```
3.  **개발 서버 실행**:
    ```sh
    npm run dev
    ```
    개발 서버가 시작되면 `http://localhost:3000`에서 애플리케이션에 접속할 수 있습니다.
